# Hedra Video Model Guide

There isn’t one “best” model. There is a best model for your goal.

If you’re unsure where to begin:

• Dialogue or presenters → Start with Hedra-native models\
• Cinematic visuals → Use text-to-video tiers\
• Copying motion → Use Motion / Reference models\
• Testing ideas → Use budget or fast tiers

---

# Hedra Native Models (Recommended Starting Point)

---

### [Hedra Avatar](https://www.hedra.com/app/video?model=hedra-avatar) _-7 credits per second_

**Category:** Audio-Driven Avatar\
**Audio Support:** Yes\
**Input Type:** Image + Audio

**Best For:**\
Talking heads, presenters, tutorials, explainers.

**Max Duration:** 10 minutes per clip

**Strengths:**\
• Reliable lip-sync\
• Stable performance delivery\
• Cost-efficient for longer dialogue\
• Longest single-clip duration of any model on Hedra

**Use When:**\
You need a host speaking clearly on camera, or you need long-form uncut video.

**Avoid When:**\
You’re building complex cinematic scenes.

# Credit Strategy Guidance

• Start with lower-cost models when testing ideas.\
• Upgrade to higher-tier models when finalizing.\
• Use Hedra-native models first for dialogue.\
• Use Motion models only when copying movement.\
• Use Premium tiers for maximum realism and narrative consistency.

| Use Case                              | Recommended Model                          |
| ------------------------------------- | ------------------------------------------ |
| On-camera dialogue, explainers, songs | Hedra Avatar                               |
| Long-form uncut video (up to 10 min)  | Hedra Avatar                               |
| Longer scenes with cuts               | Composer (combine clips from any model)    |
| Cinematic, photoreal B-roll           | Veo 3                                      |
| Fast prototyping / quick variants     | Hedra Avatar, Veo 3 Fast, or Kling 2.1 Pro |
| Action / splash / physics             | Hailou-02 Pro                              |
| Premium polished 1080p social clips   | Kling 2.1 Master                           |

<Note>
  Video AI models evolve rapidly. Each one improves often, so use this guide as directional, not permanent truth — good starting points for choosing the right fit. Always feel free to reach out to the Hedra support team if you need help.
</Note>
