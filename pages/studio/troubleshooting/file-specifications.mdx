Learn about the file requirements for uploading content to Hedra.

## File Size Limits

<Warning>
The maximum file size for any file type is **10MB**. This includes audio files as well as images.
</Warning>

## Audio Files

### Supported Formats

| Format | Supported |
|--------|-----------|
| .wav | Yes |
| .mp3 | Yes |

<Tip>
Ensure you use these file types for maximum compatibility.
</Tip>

### Audio Length Limits

| Plan | Maximum Length |
|------|----------------|
| Free | 20 seconds |
| Premium | 5 minutes |

## Image Files

### Supported Formats

- PNG
- JPG / JPEG
- WebP

### Recommendations

For best results with Character-3:
- Use clear, front-facing portraits
- Ensure good lighting
- Minimum recommended size: 512x512 pixels
- Avoid objects in hands (can cause issues in longer videos)

## Video Output

### Available Resolutions

| Resolution | Notes |
|------------|-------|
| 540p | Budget-friendly option (uses fewer credits) |
| 720p | Standard quality |
| 1080p | Available with select models (Kling 2.1 Master) |

### Aspect Ratios

- 16:9 (Landscape)
- 9:16 (Portrait)
- 1:1 (Square)

<Tip>
Match your output aspect ratio to your source image dimensions for optimal results.
</Tip>
