Hedra Studio is an integrated platform that eliminates the need to switch between multiple tools. It combines the best image, audio, and video models like Kling, Minimax, and Veo2 to help you create content fast and easily.

## Three-Step Workflow

<Steps>
  <Step title="Create an Image">
    Begin by either uploading an existing image or generating one using AI capabilities. The platform offers tools like Flux Kontext or Imagen4 for image generation.
  </Step>
  <Step title="Add Audio">
    Produce or import audio through services like ElevenLabs or personal recordings, then synchronize it with your visual content.
  </Step>
  <Step title="Generate Video">
    Leverage the platform's video generation models to create high-quality videos using Kling, Minimax, or Veo2.
  </Step>
</Steps>
