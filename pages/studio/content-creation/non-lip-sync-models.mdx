Learn how to generate videos using non-lip-sync models like Veo3 and Kling.

## Steps to Generate Video

<Steps>
  <Step title="Access Video Section">
    Navigate to the **Video** module in the interface.
  </Step>
  <Step title="Select Your Model">
    Select a model (Veo, Kling, etc.) in the model dropdown.
  </Step>
  <Step title="Input Creative Parameters">
    Provide a text prompt and/or start frame to direct the video generation process.
  </Step>
  <Step title="Adjust Settings">
    Customize parameters including:
    - Aspect ratio
    - Resolution
  </Step>
  <Step title="Generate and Refine">
    Generate your video, review the results, then download or refine selectively based on the output.
  </Step>
</Steps>

## When to Use Non-Lip-Sync Models

<CardGroup cols={2}>
  <Card title="Cinematic B-Roll" icon="film">
    Use **Veo 3** for high-quality cinematic footage
  </Card>
  <Card title="Action Sequences" icon="bolt">
    Use **MiniMax Hailou** for physics-heavy motion
  </Card>
  <Card title="High Resolution" icon="expand">
    Use **Kling 2.1 Master** for premium 1080p
  </Card>
  <Card title="Quick Prototypes" icon="clock">
    Use **Veo 3 Fast** for rapid iteration
  </Card>
</CardGroup>

<Tip>
For talking head content with lip synchronization, use [Character-3](/pages/studio/getting-started/character-3) instead.
</Tip>
