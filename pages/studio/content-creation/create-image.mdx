Learn how to generate images using Hedra's various AI models.

## Steps to Create an Image

<Steps>
  <Step title="Access Image Tool">
    Select the **Image** option from the left sidebar.
  </Step>
  <Step title="Choose a Model">
    Select a model from the dropdown (e.g., Nano Banana, Seedream 4.0). Note the generation cost and description of what each model is best at.
  </Step>
  <Step title="Configure Generation Settings">
    - Enter your text prompt
    - Select aspect ratio
    - Choose resolution
    - Set number of generations

    You can also upload reference images or use prior assets for inspiration via the **+** button.
  </Step>
  <Step title="Generate Images">
    Click **Generate** and the resulting images will appear in the space above the prompt bar.
  </Step>
  <Step title="Use Your Images">
    Hover over a generated image to access additional tools:
    - **Bottom options:** Create Video, Remix, Regenerate
    - **Upper right:** Download or Full Screen
  </Step>
</Steps>

## Available Models

Different models have varying capabilities and pricing. Check the model dropdown for:
- **Nano Banana** - Fast, efficient generation
- **Seedream 4.0** - High-quality outputs
- **Flux Kontext** - Context-aware generation
- **Imagen4** - Google's advanced model

<Note>
Available options depend on which model is selected. Some models support reference images while others may have different resolution options.
</Note>
