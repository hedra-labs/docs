---
title: 'Quickstart'
---

Getting started with the Hedra API is easy.

### Step 1: Register an API key

Go to your [profile](https://hedra.com/api-profile), generate an API key, and purchase API credits.

### Step 2: Clone the starter repo

The Hedra API starter repo contains example projects leveraging the API. Ensure you have `git-lfs installed`

```bash
git clone https://github.com/hedra-labs/hedra-api-starter
```

### Step 3: Create your first avatar

```
python main.py --api-key sk_hedra-*** --audio assets/audio.wav --img assets/1:1.jpg --ar 1:1

```
